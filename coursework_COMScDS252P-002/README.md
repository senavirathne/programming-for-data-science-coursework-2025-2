# Programming for Data Science Coursework 2025

**Course Module:** COMScDS252P  
**GitHub Repository:** [https://github.com/senavirathne/programming-for-data-science-coursework-2025-2/](https://github.com/senavirathne/programming-for-data-science-coursework-2025-2/)

This repository contains the complete submission for the Programming for Data Science Coursework. The project is divided into three main questions, covering object-oriented programming, data analysis, and AI data ethics.

---

## Project Structure

### 1. Question 1: University Management System 
An Object-Oriented Programming (OOP) implementation of a university management system built using Python.
- Features robust class hierarchies for `Person`, `Student`, `Faculty`, and `Staff`.
- Implements encapsulation, polymorphism, and property validation.
- Includes an interactive Jupyter Notebook UI utilizing `ipywidgets` for CRUD operations on departments, courses, and students.
- Calculates student GPAs and manages academic probation statuses based on credit load.

### 2. Question 2: Data Analysis
A comprehensive exploratory data analysis (EDA) of an e-commerce dataset using `pandas`, `matplotlib`, and `seaborn`.
- Conducts data cleaning, missing value handling, and feature engineering.
- Explores sales trends, customer behavior, and product performance through visualizations.
- Performs statistical hypothesis testing (A/B testing) to determine the impact of promotional campaigns.
- Trains a machine learning model to predict customer churn or sales velocity as part of advanced analytics.

### 3. Question 3: Data Ethics - AI Ethics in Healthcare
An essay examining the ethical implications of data science and AI within the healthcare sector.
- Compares data privacy regulations (HIPAA vs. GDPR) and the limits of data anonymization.
- Analyzes algorithmic bias using documented cases (e.g., Obermeyer et al., 2019) and proposes mitigation strategies.
- Develops a 6-point ethical decision framework for healthcare data scientists.
- Explores stakeholder impacts and provides policy recommendations governing clinical AI use.

### Documentation
- `technical_report.md` / `technical_report.pdf`: The complete written report detailing methodologies, findings, mathematical formulas, and analysis results for all questions.
- `requirements.txt`: Lists all Python dependencies required to run the notebooks and scripts.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/senavirathne/programming-for-data-science-coursework-2025-2.git
   cd programming-for-data-science-coursework-2025-2/coursework_COMScDS252P-002
   ```

2. **Install dependencies:**
   It is recommended to use a virtual environment.
   ```bash
   pip install -r requirements.txt
   ```

3. **Running the interactive notebooks:**
   Launch Jupyter Notebook or Jupyter Lab from the project root:
   ```bash
   jupyter lab
   ```
   Navigate to the respective directories to open the `.ipynb` files.
